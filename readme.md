# 🚀 Guia Completo: Registro e Autorização de App no Trello com Python

Este guia orienta você no processo de registrar, autorizar e usar as APIs do Trello utilizando Python, conforme as instruções e requisitos solicitados na atividade do bootcamp.

## 📋 Pré-requisitos

- Conta ativa no Trello
- Python 3.7 ou superior instalado
- Pip (gerenciador de pacotes Python)
- Navegador web
